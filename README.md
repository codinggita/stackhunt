## 💡 Overview

**Stackhunt** is an **Open Source Bounty Platform** where companies and project maintainers can post **bounties on open-source issues**, and developers can **earn rewards by solving them**.

The platform connects **organizations willing to fund development** with **developers looking for real-world problems to solve**.

---

## ❗ Problem

Open-source projects often face:

* Too many unresolved issues
* Limited developer availability
* Lack of incentives for contributors

---

## ✅ Solution

**Stackhunt** solves this by:

* Allowing companies to **fund issues**
* Helping maintainers **resolve issues faster**
* Enabling developers to **earn rewards by fixing bugs and features**

---

## ✨ Features

* 🪙 Bounty marketplace for open-source issues
* 👨‍💻 Developer profiles
* 🔍 Browse issues by technology or difficulty
* 🔗 Pull request verification
* 💰 Reward payouts after PR merge
* 🏆 Contributor leaderboard

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/stackhunt.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
